# resume-nextjs

이 레포는 과거에 사용했던 **이력서 템플릿 실험용 저장소**입니다.

현재 채용용으로 유지하는 공식 이력서 레포는 아래입니다.

- 공식 이력서 레포: https://github.com/snwlee/resume
- 이력서 사이트: https://snwlee.github.io/resume/

## 상태

- legacy / archived prototype
- 템플릿 기반으로 시작한 레포이며 현재는 메인 이력서로 사용하지 않음
- 최신 경력, 문구, 포지셔닝은 `resume` 레포에서만 관리

필요 시 레이아웃 참고용으로만 사용하고, 실제 지원 링크에는 `resume` 레포와 배포 페이지를 사용하는 것을 권장합니다.
